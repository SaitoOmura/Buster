# Buster

## 概要
OpenGLを用いて制作した3Dアクションゲームです。  
敵を回避・撃破しながらステージを進み、ゴールを目指します。

## 使用技術
- C++
- OpenGL
- SDL2
- FBXSDK

## 担当範囲
個人制作のため、全ての実装を担当しています。
- 描画処理
- カメラ制御（ズーム機能含む）
- 当たり判定
- モデル読み込み（FBX）
- プレイヤー・敵の挙動

## 技術的なポイント
- FBXSDKを用いたモデルデータの読み込みと描画処理の実装
- 描画用データと当たり判定データの分離による安定した衝突処理
- 衝突方向ベクトルを利用したノックバック処理
- 描画サイズのスケーリングによるカメラズーム機能の実装

## 操作方法
- 移動：スティック
- 攻撃：Bボタン
- ジャンプ：Aボタン

## 工夫した点
見た目と当たり判定の不整合を防ぐため、法線ベクトルや判定データを用途ごとに分離して管理しました。  
また、攻撃時のノックバック処理を実装し、プレイヤーがヒットを直感的に理解できるようにしています。
